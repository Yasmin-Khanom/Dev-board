Question-2: What is the difference between innerHTML, innerText, and textContent?

JavaScript-এ innerHTML, innerText, এবং textContent তিনটি প্রপার্টি HTML কনটেন্ট ম্যানিপুলেট করতে ব্যবহৃত হয়,
1.innerHTML:HTML কন্টেন্ট যোগ করতে বা পড়তে চাইলে innerHTML ব্যবহার করা হয়।
2.innerText:এটি শুধুমাত্র দৃশ্যমান টেক্সট return দেয়, এবং এটি Html tag না দেখিয়ে tag এর ভিতরের text গুলো কে দেখায়।
3.textContent:এটি পুরো text ফেরত দেয়, যতটুকুই থাকুক না কেন, Hidden এলিমেন্ট সহ।

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি টেকনিক যেখানে আমরা parent element এ ইভেন্ট হ্যান্ডলার যোগ করি, যাতে তার child element গুলোর ইভেন্টও হ্যান্ডেল করা যায়। এর ফলে অনেকগুলো child element এর জন্য আলাদা আলাদা ইভেন্ট লিসেনার না লাগিয়েও ইভেন্ট ম্যানেজ করা যায়।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো একটি প্রক্রিয়া যেখানে কোনো ইভেন্ট প্রথমে child element ট্রিগার হয় এবং তারপর parent element এর দিকে উপরের দিকে যেতে থাকে।
